## Descrição

Setup coverage reporting tools (Vitest + nyc/c8) para monitorar code coverage. Configurar:
- Coverage collection durante testes (unit + E2E)
- Coverage reporting (HTML, LCOV, console)
- Coverage thresholds (enforcement em CI)
- Coverage badges (display in README)

## Acceptance Criteria

- [x] Vitest configurado com `coverage` option ativa
- [x] Coverage reports gerados em `coverage/` directory
- [x] Coverage thresholds enforçados (70%+ line coverage)
- [x] CI/CD falha se coverage < 70%
- [x] Coverage badge criada (Markdown for README)
- [x] HTML coverage report acessível (coverage/index.html)

## Scope

### IN
- Setup coverage tools (vitest --coverage)
- Configure c8/nyc
- Set thresholds (70%)
- Create badge
- CI/CD integration

### OUT
- Não otimizar coverage (aceitar < 70% por enquanto)
- Não implementar "coverage-driven" tests

## Tasks

- [ ] Instalar coverage dependencies (`npm install -D vitest c8`)
