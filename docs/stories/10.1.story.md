## Descrição

Configurar framework de E2E tests usando Playwright. Setup inclui:
- Instalação e configuração de Playwright
- Base test runner com authentication
- Example test para workflow crítico (contrato creation)
- CI/CD integration (GitHub Actions hook)

## Acceptance Criteria

- [x] Playwright instalado e configurado (`npm install -D @playwright/test`)

- [x] Example E2E test: user login → create contrato workflow
- [x] CI hook configurado (testes rodam em GitHub Actions)
